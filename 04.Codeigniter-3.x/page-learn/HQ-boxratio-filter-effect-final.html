I.Cách sử dụng:
-Copy đoạn mã html bên dưới và thay đổi các class được đánh dấu màu xanh bằng tên các class trên demo để thấy được sự thay đổi của các hiệu ứng.

No Filter

Filter-black-white

Filter-one-color

Không áp dụng hướng
Chỉnh sửa : Thêm class filter-edit sau class filter,vào settingProject-System.css tìm class : ".filter-edit.filter-..." tuỳ chỉnh màu sắc theo ý muốn.
Filter-gradient-two-color

Áp dụng với hiệu ứng gradient filter 2 color

Có áp dụng hướng:to top, to bottom left ... Chỉnh sửa : Thêm class filter-edit sau class filter,vào settingProject-System.css tìm class : ".filter-edit.filter-..." tuỳ chỉnh màu sắc theo ý muốn.
Filter-gradient-three-color

Áp dụng với hiệu ứng gradient filter 3 color,có thể tuỳ chỉnh màu sắc có color với settingProject-System.css
