1.Color

HQ color được sử dụng theo quy tắc màu 60-30-10 trong thiết kế.Và không có bảng màu cố định mà phụ thuộc vào 1 source color do HQ-Multimedia dựa trên từng dự án,mục đích thiết kế

-Cách sử dụng : chỉ cần copy souce màu và paste vào phần root của file hq-color.css.

Soucre Color by HQ-Multimedia

-Trong bộ màu 60-30-10 được phân ra thành 4 dạng sau:

  • 1.Màu gốc
  • 2.Màu tương phản với màu gốc (Contrast)
  • 3.Màu tương đồng với màu gốc(Analogous)
  • 4.Màu đơn sắc từ màu gốc(Monochromatic)

-Ngoài bộ màu chính theo quy tắc 60-30-10 thì sẽ có bộ màu basic cố định.

1A.Color Background
-Màu gốc : class .color-10
-Màu tương phản : class .color-10-contrast
-Màu tương đồng màu gốc : class .color-10-analogous
-Màu đơn sắc từ màu gốc : class .color-10-monochromatic
